anime_music

Anime music searcher for Hydra modules

Usage

To depend on anime_music you will need add the jitpack.io repository in the project build.gradle/settings.gradle file:

build.gradle

allprojects {
    repositories {
        mavenCentral()
        maven { url "https://jitpack.io" }
        // ...
    }
}
